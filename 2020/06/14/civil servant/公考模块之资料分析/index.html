速算技巧

速算①

需要背诵记忆并熟练使用

速算②

一个数 X 1.5，等于这个数 + 本身的一半

速算③

一个数 ÷ 5，等于这个数 X 2，小数点前移 1 位

一个数 ÷ 25，等于这个数 X 4，小数点前移 2 位

一个数 ÷ 125，等于这个数 X 8，小数点前移 3 位

速算④

一个数 X 1.1，等于这个数错位相加

一个数 X 0.9，等于这个数错位相减

截位直除

什么截位直除？

截位就是四舍五入保留几位

如何截位？

  1. 看选项

首位：

  1. 首位不同（选项差距大，截两位）

  2. 首位相同（再看最接近的两个选项的次位差）

次位差：

  1. 次位差 < 首位（选项差距大，截两位）

如：21 和 25，次位差 5 - 1 = 4 > 2

  1. 次位差 ≤ 首位（选项差距小，截三位）

如：21 和 23，次位差 3 - 1 = 2 = 2

截谁？看算式

  1. 一步除法（A/B)：直截分母
  2. 多步除法 (A/B X C/D …..)：分子分母均截位后再约分

为节省时间，实际做题时要“少写多看”，注重截位列式，多从选项找答案

截位量级

若选项存在约 10 倍关系的时候，考虑数量级（小数点、位数、单位）

方法：

  1. 截两位（为了知道商几）

  2. 保留量级（为了知道结果是几十还是几百）

分数比较

比较分子、分母的大小方法：

  1. 一大一小：直接看，分子大的分数大
  2. 同大同小：
  1. 竖着直接除（注意量级，倍数关系明显 ⇒ 分子与分母）
  2. 横着看速度（谁快谁厉害，慢的看成 1 ）

若横看不直观能看出来谁快且倍数关系也不明显，可用增长率判别速度

内容：纯文字型材料，图表型材料

对于文字型：2步走战略

  1. 时间 + 主语 （大概20 - 30S）
  2. 只看结构（大概5S）快速定位

基期与现期

基本术语：

  1. 基期量与现期量

资料分析中常设计两个量的比较，作为对比参照的时期称为基期，对应的量称为基期量；而相对于基期的时期称为现期，所对应的量称为现期量

  1. 增长量与增长率

增长量是用来表述基期量与现期量变化的绝对量

增长率是用来表述基期量与现期量变化的相对量

  1. 同比与环比

同比：一般和上年同一时期相比较

环比：与相邻的上一个时期相比较

即，同比看年，环比看尾

表格区分如下

现期 同比 环比
2017.10 2016.10 2017.09
2017年第三季度 2016年第三季度 2017年第二季度
2017年第一季度 2016年第一季度 2016年第四季度

基期

公式：

  1. 基期量 = 现期量 - 基期量

用尾数法对比选项进行估算，没办法在进行计算

  1. 基期量 = 现期量 / ( 1+r )

r 表示增长率或减少率，| r | > 5%截位直除，| r | ≤ 5%化除为乘

化除为乘：求基期量，且选项差距小

方法：A / (1 + r) = A - A X r； A / (1 - r) = A + A X r

注意：读问题，看时间；判题型，找数据；列式子，先别算；要想快，看选项。

术语拓展：

顺差：在一个时期内，一个国家（或地区）的出口商品额大于进口商品额，叫做对外贸易顺差（又叫出超）

即 顺差 = 出口额 - 进口额

逆差：在一个时期内，一个国家（或地区）的出口商品额小于进口商品额，叫做对外贸易逆差（又叫入超）

即 逆差 = 入口额 - 出口额

基期和差：

  1. 以坑治坑：先观察大小关系排除，再看现期坑选择
  2. 截位直除：以坑治坑不好用了再截位直除

现期量

即给现在求后面某时期的值，或给过去，求现在某时期的值

公式：

现期量 = 基期量 + 增长量 = 基期量 X (1+r)

题型拓展

公式：（ 1+名义增长率）= （ 1+实际增长率）×（1+CPI）

化简得：名义增长率 = 实际增长率 + CPI + 实际增长率 × CPI

一般增长率

基本术语;

  1. 增长率

增长率是表述现期量与基期量变化的相对量。增长率又称为增速、增幅或者增长幅度、增值率等，增长率为负时表示下降，下降率也可以直接写成负的增长率

  1. 百分数与百分点

⑴ 百分数是用来反映量之间的比例关系

⑵ 百分点是用来反映百分数变化

  1. 增长率与倍数

⑴ 增长率指比基数多出的比率，倍数指两数直接的比值

⑵ 若 A 是 B 的 n 倍，则 n = r + 1

  1. 成数与翻番

⑴ 成数：几成相当于十分之几

⑵ 翻番：翻一番为原来的 2 倍，翻两番为原来的 4 倍，依此类推，翻 n 番为原来的 2^n^ 倍

  1. 增幅、降幅与变化增幅

⑴ 增幅一般就是指增长率，有正负之分

⑵ 降幅指下降的幅度，降幅比较大小时，前提必须为下降

⑶ 变化幅度指增长或下降的绝对比率，变化幅度比较大小时用增 / 降幅的绝对值

一般增长率计算

题型识别：增长 / 下降 + % / 成 / 倍

形式：

  1. 给百分点：直接加减
  2. 无百分点：r = 增长量 / 基期量

易错点：高减低加

2017 年收入 10 万元，同比增长 10%，增速比去年提高 / 降低 5 个百分点，则 2016年的增长率为：

10% - 5% = 5%（提高）；10% + 5% = 15%（降低）

2017 年收入 10 万元，同比下降 10%，增速比去年提高 / 降低 5 个百分点，则 2016年的增长率为：

-10% - 5% = -15%（提高）；-10% + 5% = -5%（降低）

2017 年收入 10 万元，同比下降 10%，增速比去年扩大 / 收缩 5 个百分点，则 2016年的增长率为：

-( 10% - 5% ) = -5%（扩大）；-( 10% + 5% ) = - 15%（收缩）

综上，出现增速直接运算（注意增长量有正负之分），出现降幅或增幅（扩大或收缩），先数值运算后再带符号

公式：( 1+r )^n^ = 现期量/基期量

混合增长量

线段法口诀：

  1. 混合之前写两边，混合之后写中间
  2. 距离和量成反比

  1. 混合后居中 (整体增长率介于部分增长率之间)

资料分析

增长量

基本术语：

  1. 增长量是用来表述基期量与现期量变化的绝对量，增长率则是表述两者变化的相对量
  2. 年均增长量 = （现期量 - 基期量）/ 年份差

增长量计算

题型识别：增长 + 具体单位（人 / 元 / 吨 等单位）

公式：

增长量 = 现期量 - 基期量 = 基期量 × r = 现期量 / ( 1 + r ) × r

推论为：r 表示增长率，利用百化分将 a% 转化为分数形式 1 / n

则 现期量 / ( 1 + r ) × r = 现期量 / ( n + 1 ) 表示增长量的计算

​ 现期量 / ( 1 - r ) × r = 现期量 / ( n - 1 ) 表示减少量的计算

百化分记忆表如下图：

百化分的多个使用方式

  1. 整数%，即N%，利用 1 / N` 来记忆，如 19% = 1 / 5.3
  2. 增长率百化分之放缩法 如 2.5% = 1 / 40 则 25% = 1 / 4
  3. 增长率百化分之取中法

即若遇到百分数左右难取舍且选项接近，取中即可

如

20% = 1 / 5 16.7% = 1 / 6 推出 18.5% = 1 / 5.5 或 5.5% = 1 / 18.5

16.7% = 1 / 6 14.3% = 1 /7 推出 15.4% = 1 / 6.5 或 6.5% = 1 / 15.4

年均增长量

题型识别：年均 (平均每年) + 增长 + 单位

公式：年均增长量 = （现期量 - 基期量）/ n(年份差)

补充之年份差的判定

  1. 一般情况（江苏除外）2011年 - 2015年，年份差 n = 4 ，基期为2011年，现期为2015年
  2. 五年规划（国家规定）“十二五”期间，年份差 n = 5 ，基期为2010年，现期为2015年

“十一五”规划：2006 - 2010；“十二五”规划：2011 - 2015；“十三五”规划：2016 - 2020

增长量比较

题型识别：增长最多 / 最少

形式：

  1. 给现期量、基期量：增长量 = 现期量 - 基期量

  2. 给现期量、增长率r

    ⑴ 大大则大：现期量大，同时 | r | 也大，则其增长量 / 减少量大
    ⑵ 一大一小：百化分，转化为分数比较

做题时先用大大则大排除，再用百化分进行比较

现期比重

题型识别：…占…的比重

公式：比重 = 部分 / 整体

速算方法：截位直除

补充：

比重图，以饼状图的形式表示部分占整体的比重

饼图看图的方法：

一般为从 12 点钟方向沿顺时针方向，按照图例顺序依次排列，注意找特殊值 (1/2，1/4等)并判断内部比例关系

基期比重

题型识别：问题的时间在材料之前，…占…的比重

公式：( A / B ) × ( 1 + b ) / ( 1 + a )

其中，A为部分现期，B为总体现期，a 为部分增长率，b 为总体增长率

速算方法：

  1. 截位直除
  2. 先算现期比重 A / B ，再看 ( 1 + b ) / ( 1 + a ) 大于还是小于 1 ，再对比选项

两期比重

题型识别：有两个时间 + 比重

公式：

A / B 是现期比重，a 为部分增长率，b 为整体增长率

则，a > b，比重上升；a < b，比重下降；a = b，比重不变

最题技巧：

  1. 先判方向：正升负降（ a - b 的正负 ）
  2. 定大小：比重必小于| a- b |
  3. 无法全部排出选项时，看下 A / B

平均数

基本术语：

  1. 平均数指多个数的平均值，即多个数的和除以数的个数；也可以指二者的比例。如收入平均值 = 收入 / 人数

现期平均数

题型识别：问题时间与材料一致 + 平均 (每 / 均 / 单位)

公式：平均数 = 总数 / 个数 = A / B

计算形式：后除以前

如 人均收入 = 收入 / 人；每亩产量 = 产量 / 亩数；单位面积产量 = 产量 / 面积

速算技巧：截位直除

平均数算法：削峰填谷

  1. 定基准，算差距
  2. 汇总除以个数再加上基准

如：86、88、90、91、85

基准可以定为87(基准一般尽量取中位数附近)

则 87 + ( -1 + 1 + 3 + 4 + -2 )/5 =87 + 1 =88

即平均数为 88

基期平均数

题型识别：问题时间在材料之前 + 平均 (每 / 均 / 单位)

公式：基期平均数 = A/B × ( 1+b )/( 1+a )

速算技巧：截位直除；与现期进行比较对比选项

两期平均数 (平均数的增长量)

题型识别：题目中涉及2个时间 + 平均数问法

平均数 + 增长 + 单位

公式：现期平均 - 基期平均 = A/B - A/B × ( 1+b )/( 1+a ) = A/B × ( a-b )/( 1+a)

a > b，平均数上升；a < b，平均数下降；a = b，平均数不变

注意：与比重进行区分 a 和 b

平均数的增长率

题型识别：平均数 + 增长率 + %

公式：r = ( a-b )/( 1+b )

其中，a 为分子增长率，b 为分母增长率

做题逻辑：

  1. 先确定分子 a，分母 b
  2. 带公式看选项

倍数

基本术语：

  1. 倍数用来表示二者的相对关系
  2. 若 A 是 B 的 n 倍，则 n = A/B
  3. 若 A 比 B 高(多) n 倍，则 n = A/B - 1
  4. 现期倍数：A/B ；基期倍数：A/B × ( 1+b )/( 1+a)

注意区分提问中的 “是” 还是 “多(高)” n 倍

注：A 超过 B 的 n 倍：A > B × n
